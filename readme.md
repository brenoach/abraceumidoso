💜 Abrace um Idoso: Conectando Gerações com Carinho Status do Projeto: 🚀 Em desenvolvimento (MVP concluído)

O Abrace um Idoso é uma plataforma digital desenvolvida para combater o isolamento social da terceira idade em casas de repouso e instituições de longa permanência. O sistema atua como uma ponte inteligente entre voluntários dispostos a doar tempo e idosos que buscam companhia, transformando a gestão de visitas em uma experiência moderna, segura e afetiva.

🌟 O Ineditismo do Projeto Diferente de sistemas de gestão hospitalar ou administrativa, o Abrace um Idoso foca no capital humano.

Curadoria de Afeto: Inspirado na interface fluida de plataformas de streaming (Netflix Style), o voluntário não apenas escolhe um horário; ele escolhe uma história. A vitrine de residentes humaniza o processo de agendamento.

Gestão de Disponibilidade Dinâmica: Instituições podem liberar janelas de horários específicas de acordo com a rotina de cada idoso, garantindo que o voluntário chegue no momento em que ele é mais necessário.

Inclusão Digital para o Terceiro Setor: Digitaliza processos que antes eram manuais (anotações em cadernos ou telefonemas), permitindo um controle rigoroso de quem entra e sai das instituições.

📈 Escalabilidade e Visão de Futuro O projeto foi construído com uma arquitetura modular que permite um crescimento acelerado:

Modelo Multi-Tenant (SaaS): A base de dados e a estrutura de pastas estão preparadas para evoluir de uma única instituição para uma rede de casas de repouso.

Expansão para Ecossistema Mobile: A interface construída com Flexbox e Grid facilita a transição para um aplicativo nativo (React Native), permitindo notificações push para lembrar voluntários de suas visitas.

Métricas de Impacto Social (ESG): O sistema gera dados valiosos sobre horas de voluntariado e frequência de visitas, informações fundamentais para relatórios de impacto social e captação de recursos para as instituições.

🛠️ Stack Tecnológica O projeto utiliza tecnologias robustas e de ampla comunidade, garantindo fácil manutenção:

Linguagem: PHP 8.2+

Banco de Dados: MySQL (Relacional)

Front-end: HTML5, CSS3 (Moderno), JavaScript

Segurança: Autenticação via Sessão, Recuperação de senha com tokens criptografados e PHPMailer.

UI/UX: Design responsivo focado em acessibilidade.

🚀 Principais Funcionalidades [x] Painel do Voluntário: Visualização de idosos disponíveis e histórico de agendamentos.

[x] Agendamento Inteligente: Validação de horários permitidos e status de visitas (Pendente/Confirmado).

[x] Recuperação de Acesso: Sistema de redefinição de senha via e-mail corporativo.

[x] Fallback de Imagens: Gerador de avatares automáticos para garantir a identidade visual mesmo sem fotos no servidor.
